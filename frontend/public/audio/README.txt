AUDIO FILE LOCATION
===================

Place your AI-generated sales insights podcast audio file here with the name:
sales-insights-podcast.mp3

This file will be played by the InsightsPodcastPlayer component on the Home page.

For best results:
- Format: MP3
- Bitrate: 128kbps or higher
- Length: 2-10 minutes recommended

The audio file should contain:
- AI-generated analysis of the company
- Sales opportunities and insights
- Market analysis and recommendations
- Key decision maker information

Example filename: sales-insights-podcast.mp3
